tabcomplete

Type something and then complete the words by pressing tab. Go ahead and try!

Press up ↑ or down ↓ to go through matches.

Press left ← to cancel after auto-completion.

Plugin usage 

new Tabcomplete(input|textarea, wordsArray [, options]);

Default options 

var options = {
	after: "",
	arrowKeys: true,
	hint: "placeholder",
	match: function(word, words) { .. },
	caseSensitive: false,
	minLength: 1,
	wrapInput: true,
	complete: 'tab'
};

Example 

<script src="tabcomplete.js"></script>
<script>
	window.addEventListener('load',function(){
		new Tabcomplete(document.getElementbyId('target'),[
			"array",
			"with",
			"words",
			"to",
			"complete"
		]);
	});
</script>
